Introduction

Callme is a customer engagement tool, based on the vidyo.io platform, that you can add to any website. You can be easily video chatting with your website visitors with a few snippets of code. For example, when a customer clicks a button to speak with an agent, you (the agent) will be messaged via email or SMS of such a request. You click the link in the message and you can start video chatting immediately. You are just a few steps away from engaging with your customers live via video chat within your very own website!

Getting Started

Before we get started we need a little bit of information to configure your first callme application:

  1. Grab an Application ID and Developer Key from your API Key's. They are used to create tokens automatically and will always remain hidden.
  2. Create a call me configuration. As you enter the requested information on each field of the form, follow the instructions in the popup.
  3. Try out the demos once you have a configuration and tweak your configuration before you go live.
  4. From the list of call me configurations, note the callme App Key as this what will be used to identify the callme configuration on your website.
  5. Follow the instructions in the HTML comments below to setup your website and start video chatting with your customers:
<!DOCTYPE html>
 <html>
     <head>
         <!-- add this script to your webpage wherever call me functionality is desired -->
         <script src="https://callme.app.vidyo.io/js/VidyoIOAgent.js"></script>

         <!-- the vidyoIOAgent method that will start a call -->
         <script type="text/javascript">
             function callMe() {
                 var config = {
                     appUrl: "https://callme.app.vidyo.io",
                     appKey: "hnq32BAs",        // set your appKey here
                     viewId: "vidyoIOAgentView" // set your div where video will be seen
                 };
                 vidyoIOAgent(config);
             }
         </script>
     </head>
     <body>
         <!-- create a button to invoke a method to get the call started -->
         <button onclick="javascript:callMe()">Call Me</button>

         <!-- the div where the video session is displayed -->
         <div style="position: absolute; bottom: 0; right: 0;">
             <div id="vidyoIOAgentView" style="float: right; width: 854px; height: 480px;"></div>
         </div>
     </body>
 </html>

or a more complicated example where a YouTube video is displayed while the customer is waiting for an agent:

<!DOCTYPE html>
 <html>
     <head>
         <!-- add this script to your webpage wherever call me functionality is desired -->
         <script src="https://callme.app.vidyo.io/js/VidyoIOAgent.js"></script>

         <!-- the vidyoIOAgent method that will start a call -->
         <script type="text/javascript">
             function callMe() {
                 var config = {
                     appUrl: "https://callme.app.vidyo.io",
                     appKey: "hnq32BAs",         // set your appKey here
                     viewId: "vidyoIOAgentView", // set your div where video will be seen
                     onUserJoined : function() {
                         // start a YouTube video while waiting for agent
                         document.getElementById("vidyoIOAgentView").style.display = "none";
                         document.getElementById("videoFrame").src="https://www.youtube.com/embed/7VENzZpJCec?autoplay=1";
                     },
                     onAgentJoined : function() {
                         // end YouTube video when agent joins
                         document.getElementById("videoFrame").src="";
                         document.getElementById("vidyoIOAgentView").style.display =  "block";
                     },
                     onAgentLeft : function() {
                         console.log("agent left");
                     },
                     onCallFailure: function() {
                         alert("call failed");
                     }
                 };
                 vidyoIOAgent(config);
             }
         </script>
     </head>
     <body>
         <!-- create a button to invoke a method to get the call started -->
         <button onclick="javascript:callMe()">Call Me</button>

         <!-- the div where the video session is displayed -->
         <div style="position: absolute; bottom: 0; right: 0;">
             <div id="vidyoIOAgentView" style="float: right; width: 854px; height: 480px;"></div>
         </div>

         <!-- the div for the YouTube video -->
         <div style="position: absolute; bottom: 0; right: 0;">
             <iframe id="videoFrame" style="float: right; width: 854px; height: 480px;" src="" frameborder="0"></iframe>
         </div>
     </body>
 </html>

Notes

  • Callme works on browsers that support WebRTC and plugins only.
  • Your vidyo.io Application ID and Developer Key are kept private at all times. The only thing visible in Javascript code is your callme App Key. Misuse of your App Key is prevented via Referrer Header checking. It is strongly recommended to setup Referrer checking by filling out the hostname of your server or servers where you will enable callme.
  • If you need more features than those available, this application is open-source and you can modify it as you need on GitHub.
