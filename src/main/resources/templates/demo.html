This is a basic demo for a customer requesting to speak with an agent.



When the customer presses the Call Me button, you will instantly get an email and/or SMS message which you configured, saying a customer is requesting a video chat. That message contains a link to start the video chat with the customer. While the customer is waiting for you to join, they will see a message "Waiting for agent."

When you try out the demo, please ensure you are a customer on one computer and an agent on another computer.



Be sure to check out the source code of this page (How?).

In the source, you will notice we have filled in your appKey for this demo, so adding the same functionality to your website is a matter of a copy and paste!
